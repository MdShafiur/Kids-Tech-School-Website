Micro:bit

Let's learn coding using Micro:bit! Micro:bit is a pocket-sized computer that introduces you to how software and hardware work together. It has an LED light display, buttons, sensors and many input/output features that, when programmed, let it interact with you and your world!

Scratch

Scratch is a drag-and-drop programming language for children designed to create things such as games, animations, art, and stories. It allows them to explore and experiment with the ideas of computer programming by using "blocks" that are put together to make simple code.
